# KSE-codebase

The shared code space for **Kyiv School of Economics** — automations, prompts, and internal tooling, in the open (within KSE), so teams can share ready-made solutions instead of rebuilding them.

We're moving toward being **AI-integrated, AI-native, and data-driven**. This org is where the automations live.

## Units

Work is grouped by unit. Each unit is a **Team** here — membership in a team grants access to that unit's repositories. Browse them at [Teams](https://github.com/orgs/KSE-codebase/teams).

| Unit | Team |
|---|---|
| President Office | `president-office` |
| Business School | `business-school` |
| University | `university` |
| ProfTech | `proftech` |
| KSE Institute | `kse-institute` |
| Charity Fund | `charity-fund` |

## How it works

- **Each automation is its own repository**, owned by its maintainer, who can change it freely.
- **Repos are shared by default** so others can learn from and reuse them; sensitive ones can be private or restricted.
- **No secrets in code** — GitHub Secrets or Apps Script Script Properties.
- Every repo has a README you can follow end-to-end.
